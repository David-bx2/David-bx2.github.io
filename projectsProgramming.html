Los Mejores Autos


Este proyecto fue desarrollado como trabajo final del programa Certified Tech Developer de Digital House,
aplicando conocimientos adquiridos durante el curso en desarrollo frontend, backend, arquitectura de software, pruebas y diseño responsivo.
Se trata de una plataforma web de reservas de vehículos, que permite a los usuarios buscar, visualizar y reservar autos según disponibilidad, ubicación y características.
El sistema incluye funcionalidades completas tanto para usuarios como para administradores, abarcando desde la autenticación hasta la gestión de productos y reservas.

Tecnologías y herramientas utilizadas:

    Frontend
  • Lenguaje: JavaScript
  • Framework: React
  • Librerías: Axios, React Router, React Date Range, React Icons, Bootstrap
  • Estilos: CSS personalizado y Bootstrap
  • Notificaciones: React Hot Toast

      • Backend
  • Lenguaje: Java
  • Framework: Spring Boot
  • Base de datos: H2 (en memoria)
  • ORM: JPA con Hibernate
  • Control de dependencias: Maven
  • Servicios adicionales: Envío de correos vía Spring Mail
